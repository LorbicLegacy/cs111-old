Reading The Blog

This blog is written by Vikas Patel and contains posts about many programming languages like: C, C++, JAVA, Python, C# etc. and other technologies.

"So, start reading the blog."


So, you want to learn programming. Welcome to one of the great adventures of the twenty-first century. Programming requires little in the way of specialized equipment; the software tools can all be downloaded for free off the Internet, and it can be practiced in the safety and comfort of your own home, without having to ask anyone’s permission. This blog will ease you in gently by introducing you to the software you will need to create your programs: a command-line interface, which allows you to use programming in interactive mode, and a text editor for writing scripts—nothing more complicated than that.

I will also show you where to go to find help and documentation, so you can decode the sometimesimpenetrable jargon that seems to surround this, the geekiest of all technical disciplines. To begin with, you will need to make sure that you have a decently recent version of compiler installed on your machine or follow the steps later in this blog to install it. This blog explains how to make sure that you have everything set up correctly and that you have suitable references at hand before you start your journey.
